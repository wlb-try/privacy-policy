# 隐私政策

**生效日期：2026年1月6日**

**应用名称：** 内在价值

本隐私政策适用于 **内在价值**（以下简称“本应用”）。我们尊重并保护您的隐私。以下是关于本应用数据处理的完整说明：

## 1. 本应用不收集任何用户数据
- 本应用 **不收集、不存储、不传输** 任何个人数据或设备信息。
- 本应用 **完全离线运行**，**不连接互联网**，不会向任何服务器发送数据。
- 您在使用本应用过程中产生的所有信息均仅保存在您的设备本地，不会被上传或共享。

## 2. 无第三方 SDK 或服务
- 本应用 **未集成任何第三方 SDK**（如广告、分析、崩溃报告、社交媒体等）。
- 不存在 Google Firebase、Facebook SDK、AdMob、AppLovin、友盟、微信 SDK 或其他类似服务。

## 3. 无追踪或分析功能
- 本应用 **不包含任何形式的用户行为追踪、数据分析或遥测功能**。
- 无 Cookie、无 IDFA、无设备标识符读取。

## 4. 权限使用说明
- 本应用 **不请求任何敏感权限**（如位置、相机、麦克风、联系人、照片等）。
- 如系统因功能需要临时请求权限（例如文件选择），仅在用户主动操作时触发，且数据不会离开您的设备。

## 5. 隐私承诺
我们承诺：**您的数据始终属于您自己**。本应用的设计原则是“零数据收集”，确保最大化的隐私与安全。
